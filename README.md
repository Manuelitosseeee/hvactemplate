# HVAC Pro — Template sito aziendale

Template moderno e responsive per azienda HVAC. Contiene:
- index.html — pagina principale
- styles.css — stili moderni e animazioni
- scripts.js — script per animazioni, menu e form demo

Come usare:
1. Scarica i file in una repository (es. `hvactee`).
2. Personalizza testi, immagini e colori in `styles.css` e `index.html`.
3. Deployment:
   - GitHub Pages: crea un repo pubblico e vai in Settings → Pages → Branch `main` → / (root). Merge i file in `main`. La pagina sarà `https://<tuo-username>.github.io/<repo>/`.
   - Vercel / Netlify: collega il repo e deploya. Supportano deploy automatici ad ogni push.
4. Per form reali: aggiungi un backend o usa servizi come Formspree, Netlify Forms, o integrazione con email/CRM.
5. Font e immagini: uso Google Fonts e immagini Unsplash (usa le tue immagini per produzione).

Suggerimenti per personalizzazione:
- Cambia palette in :root (variables).
- Aggiungi logo reale nella header.
- Per migliori performance, ottimizza immagini e usa lazy-loading per immagini pesanti.
- Per SEO: aggiungi meta tags Open Graph e JSON-LD.
